How to create a route and view

  • 1. // routes/index.js router.get('/about', (req, res) => res.render('about', { title: "About Us" });
  • 2. // views/about.ejs <h1><%= title %></h1>
  • 3. If you download a CSS framework, then put it under public/. Then in ejs file <link rel="stylesheet" href="/stylesheets/style.css"><link/>
  • 4. Create controllers/indexController.js exports.homePage = (req, res) => { res.render('about', { isActive: 'about', }); };
  • 5. // Change routes/index.js router.get('/about', indexController.homePage);

How to create partials

  • 1. Under views/ create new folder called partials/ /views/partials/
  • 2. Create header.ejs file /views/partials/header.ejs
  • 3. Create footer.ejs file /views/partials/footer.ejs
  • 4. Cut html code starting from the top down to (and including) the body tag and paste it in header.ejs <!DOCUMENT html5> ... <body>
  • 5. // views/index.ejs <% include partials/header %>
